python基本类型：数字类型（整形,浮点型、布尔类型、复数）、10,2,8,16进制与转换、字符串及运算。

什么是代码？

代码是现实世界事物在计算机世界中的映射。

什么是写代码？

写代码是将现实世界中的事物用计算机语言来描述。

好记性不如烂笔头！

python中，一切皆对象。对象的三大特征：值 value、身份 id、类型 type。

1、数字类型 Number（整形,浮点型、布尔类型、复数）

整数：int，浮点数：float。

>>> type(1)
class 'int'
>>> type(-1)
class 'int'
>>> type(1.1)
class 'float'
>>> type(1+0.1)
class 'float'
>>> type(1+1)
class 'int'
>>> type(1+1.0)
class 'float'


>>> type(1*1)
class 'int'
>>> type(1*1.0)
class 'float'
>>> type(2/2)
class 'float'
>>> type(2//2) #//双斜杠是整除，只会保留整数部分 
class 'int'
>>> 2/2
1.0
>>> 2//2
1
>>> 1//2
0

//双斜杠是整除，只会保留整数部分

数字布尔类型 bool

bool布尔类型：表示真 True、假 False。注意首字母大写。只要是非0数字都表示bool True。

>>> type(True)
class 'bool'
>>> type(False)
class 'bool'
>>> int(False)
0
>>> int(True)
1
>>> bool(0)
False
>>> bool(1)
True
>>> bool(2)
True

除了0，None外，一系列的空值都会认为是False。

>>> bool('abc')
True
>>> bool('')
False
>>> bool([1,2,3])
True
>>> bool([])
False
>>> bool({1,2,3})
True
>>> bool({})
False
>>> bool(None)
False

复数 complex

python中表示复数，使用j。

>>> 36j
36j

2、10,2,8,16进制与转换

10进制，满10进1。0,1,2,3.。。9,10

2进制，满2进1。0,1,10

8进制，满8进1. 0,1,2.。。7,10

16进制，满16进1. 0,1,2，。。。9，A,B,C,D,E,F

60秒=1分钟也是一种进制方式，只不过是在生活中常用。

2进制在计算机中，用0b表示

>>> 0b10
2
>>> 0b11
3

8进制在计算机中，用0o表示

>>> 0o10
8
>>> 0o11
9

16进制在计算机中，用0x表示

>>> 0x10
16
>>> 0x11
17
>>> 0x1F
31

进制转换

任意进制转2进制：

>>> bin(10)
'0b1010'
>>> bin(0o7)
'0b111'
>>> bin(0xE)
'0b1110'

任意进制转10进制：

>>> int(0b111)
7
>>> int(0o77)
63
>>> int(0x7E)
126

任意进制转16进制：

>>> hex(888)
'0x378'
>>> hex(0o777)
'0x1ff'

任意进制转8进制：

>>> oct(0b111)
'0o7'
>>> oct(0x777)
'0o3567'

原始字符串 r

>>> print('c:\northwind\northwest')
c:
orthwind
orthwest
>>> print('c:\\northwind\\northwest')
c:\northwind\northwest
>>> print(r'c:\northwind\northwest')
c:\northwind\northwest

不是一个普通的字符串，而是一个原始字符串。

4、字符串的运算

字符串拼接

>>> "hello"+"world"
'helloworld'
>>> "hello"*3
'hellohellohello'

字符串裁剪

>>> "hello world"[0]
'h'
>>> "hello world"[7]
'o'
>>> "hello world"[11]
Traceback (most recent call last):
  File "", line 1, in 
IndexError: string index out of range
>>> "hello world"[10]
'd'
>>> "hello world"[-1]
'd'
>>> "hello world"[-2]
'l'

[n]表示，这个字符的序号，序号从0开始。

[-n]表示，在这个字符串的末尾，往前数n次，得到的这个字符。

>>> "hello world"[0:4]
'hell'
>>> "hello world"[0:5]
'hello'
>>> "hello world"[2:5]
'llo'
>>> "hello world"[0:-1]
'hello worl'
>>> "hello world"[0:-3]
'hello wo'

[m:n]表示，m表示开始的序号，n表示结束的序号，再加1

[m:-n]表示，m表示开始的序号，-n表示步长

[m:]表示，m表示开始的序号，什么也不写，代表截取到字符串的末尾

[-n:]表示，-n表示在这个字符串的末尾，倒着截取

>>> "hello world"[6:11]
'world'
>>> "hello world"[6:1000]
'world'
>>> "hello world"[6:]
'world'
