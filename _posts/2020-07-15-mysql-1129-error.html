错误：1129; Host is blocked because of many connection errors; unblock with 'mysqladmin flush-hosts'

原因：同一个ip在短时间内产生太多（超过mysql数据库max_connection_errors的最大值）中断了数据库连接而导致的阻塞。

解决方案1：登录mysql后执行下列命令。

show variables like '%max_connection_errors%'; #查看最大值
set global max_connect_errors = 1000; #重新设置最大值

解决方案2：命令行或终端执行下列命令。

mysqladmin -u root -p flush-hosts

接着输入root账号密码即可。
