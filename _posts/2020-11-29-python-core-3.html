代码的基本逻辑结构，包括条件控制（if else）、循环控制（for in 、while）、表达式与运算符的优先级。

1、什么是表达式？

表达式（Expression）是运算符（operator）和操作数（operand）所构成的序列。

>>> 1 + 1
2
>>> a = 3
>>> a = 1 + 2 + 3

2、表达式的优先级

>>> a = 1
>>> b = 2
>>> c = 3
>>> a + b * c
7
>>> a or b and c
1
>>> a = 1
>>> b= 2
>>> c = 3
>>> not a or b + c == c
False
>>> not a or (b + c) == c
False
>>> not a or ((b + c) == c)
False
>>> (not a) or ((b + c) == c)
False

while循环与使用场景

while的条件，不能是常量。

使用场景，你要达到一个目标，但是你也不知道需要多久，可以使用while，直到目标达成，才结束循环。

递归使用while是最好的，其他情况，建议使用for。

condition = 0

while condition < 10:
	condition += 1
	print(condition)
else:
	print('终于打到了最强王者')

for与for-else循环

使用场景，主要是用来遍历/循环 序列或者集合、字典。

a = ['apple', 'orange', 'banana']
for x in a:
	if x == 'orange':
		break # 跳出一层循环
		# continue 
	print(x)
else:
	print('fruit is gone')


b = [['apple', 'orange', 'banana'], (1,2,3)]
for x in b:
	for y in x:
		print(y)
else:
	print('fruit is gone')

for与range

使用场景：知道循环次数。

for x in range(0,10):
	print(x)

for x in range(0,10,2):#2表示步长
	print(x, end='|')
	# 0|2|4|6|8
print('')
for x in range(10,0,-2):
	print(x, end="|")
	#10|8|6|4|2
